# Images Folder

Place your birthday photos here!

## Instructions

1. Add your images to this folder (`public/images/`)
2. Name them:
   - `photo1.jpg` (or .png)
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`

3. You can add more images by updating the `images` array in `src/components/ImageSlideshow.jsx`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Recommendations

- Recommended size: 1920x1080 or larger
- Aspect ratio: 16:9 or 4:3 works best
- File size: Keep under 2MB per image for best performance

The slideshow will automatically display these images after you cut the cake!
